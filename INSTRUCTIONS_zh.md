# KVision 用户手册

> [!注意]
> 本使用说明书已在人工智能 (AI) 的帮助下自动翻译和格式化。

**KVision** 是一款先进的 VMS（视频管理系统）级桌面应用程序，设计用于同时查看实时视频流 (RTSP/ONVIF) 并与 Hikvision NVR/DVR 录像机集成（实时模式和回放存档模式）。

该程序已针对稳定性、流畅运行 (60 FPS) 和最小系统资源消耗进行了优化。

## 2. 安装与启动

### 在 Arch Linux (Pacman) 上安装
要从准备好的二进制包安装程序，请转到 `packaging/arch/` 目录并运行：
```bash
sudo pacman -U kvision-2.6.3-1-x86_64.pkg.tar.zst
```
该软件包会自动将程序、`.desktop`激活文件以及所需的海康威视SDK库安装到系统路径`/usr/lib/kvision`。

### 手动编译（来自源代码）
如果您想手动编译程序（例如，在另一个 Linux 发行版上）而不是使用现成的包：

1. 使用包管理器安装所需的构建和运行时依赖项。对于 Arch Linux / CachyOS：
   ```bash
   sudo pacman -S base-devel cmake qt5-declarative qt5-multimedia qt5-quickcontrols qt5-quickcontrols2 qt5-svg qt5-graphicaleffects qt5-tools ffmpeg git qt5-wayland
   ```
2. 使用 CMake 配置项目：
   ```bash
   cmake -B build -S . -DCMAKE_BUILD_TYPE=Release -DCMAKE_INSTALL_PREFIX=/usr
   ```
3. 编译代码：
   ```bash
   cmake --build build -j$(nproc)
   ```
4. 在系统中安装应用程序：
   ```bash
   sudo cmake --install build
   ```

### 启动中
该程序可以从系统菜单或通过在终端中键入来启动：
```bash
kvision
```

### 系统扩展故障排除 (KDE Plasma)

从版本 2.4.4 开始，KVision 通过在 Qt 中强制执行本机高 DPI 直通策略，自动检测并正确应用 KDE Plasma 设置中配置的分数桌面缩放（例如 125%、145%、150%）。用户界面现在应该可以开箱即用地正确缩放，无需任何手动干预。

#### Native Wayland 支持

为了确保 Wayland 下的最佳扩展体验和性能，请确保您的系统上安装了可选的 `qt5-wayland` 软件包。这使得 KVision 能够在 Wayland 上本地运行，而不是退回到 XWayland。

对于 Arch Linux / Manjaro / CachyOS：
```bash
sudo pacman -S qt5-wayland
```

对于 Debian / Ubuntu / Linux Mint：
```bash
sudo apt install qtwayland5
```

#### 手动缩放覆盖（旧版/后备）

如果您运行的是旧版本、不寻常的配置，或者只是希望手动强制使用与系统其他部分不同的缩放因子，您仍然可以使用环境变量覆盖应用程序缩放。

例如，要强制 **150%** 缩放，请使用以下命令从终端启动应用程序：
```bash
env QT_FONT_DPI=96 QT_SCALE_FACTOR=1.5 kvision
```

*（您可以通过在 KDE 菜单编辑器中编辑应用程序启动器 `kvision.desktop` 并将这些变量添加到“环境变量”字段来永久应用此设置）。*

## 3. 管理NVR/DVR录像机

配置与海康录像机的连接：
1. 打开选项窗口并转到**记录器**选项卡（服务器图标）。
2. 输入设备访问详细信息：
   * **IP地址**：录像机的网络地址。
   * **端口**：SDK网络端口（默认为`8000`）。
   * **用户名**：用户名（例如`admin`）。
   * **密码**：访问录音机的密码。
3. 单击**连接并发现**（或**保存并更新**）。
4. 连接成功后，应用程序将自动检测连接到 NVR/DVR 的所有活动摄像机（通道）并将其添加到列表中。
5. 单击**生成网格**按钮将自动创建一个视口布局（预设），其中包含该 NVR 中采用最佳网格布局的所有活动摄像机。
6. **显示摄像机列表 (NvrCamerasWindow)**：单击任何录像机卡上的计算机监视器图标按钮将打开一个专用窗口，将所有检测到的摄像机通道显示为交互式图块。
7. **生成缩略图（生成缩略图）**：在 NVR 摄像机列表窗口中，可以使用*“生成缩略图”*按钮。单击它会命令应用程序从后台的每个通道的子流中提取单个帧，并将它们设置为平铺背景缩略图。这提供了每个摄像机的快速视觉预览，而无需启动完整的实时反馈。
8. **点击添加功能**：应用程序不支持将图块从摄像机列表窗口拖动（拖放）到主网格。摄像机分配以简单可靠的方式完成：首先左键单击主屏幕网格中的任何视口图块将其选中（它将以明亮的边框突出显示），然后单击 NVR 摄像机列表窗口中所需摄像机图块上的绿色**“+”（分配到活动视口）** 按钮。流将立即加载到该插槽中。
9. **SDK会话状态（点指示器）**：列表中每个记录器的IP旁边，有一个彩色状态点：
   * **绿色（已登录）**：表示活动的 Hikvision SDK 会话，这是 PTZ 控制、时间线存档请求和下载所必需的。
   * **红色（未登录）**：当前未建立活动的 SDK 会话（例如，在第一个 SDK 请求之前或手动/自动注销之后）。请注意，摄像头 RTSP 实时信号将继续独立于 SDK 会话状态进行播放。
10. **本地摄像机重命名**：在 NVR 摄像机列表中的任意摄像机图块上，单击**编辑**（铅笔）图标。这将打开一个对话框，允许您为相机设置自定义名称。该名称保存在本地，并在实时播放器、时间线和图块视图中即时更新，而无需修改物理 NVR 设备。您可以随时将其重置为默认名称。
11. **从列表中删除记录器**：列表中每个配置的记录器旁边都有一个红色垃圾桶按钮。单击它会启动一个两阶段的安全协议，旨在防止意外删除：
    * **第 1 步（确认）**：出现标题为“确认 NVR 删除”* 的对话框，询问您是否确定要删除录像机。
    * **第 2 步（警告）**：出现第二个标题为“警告！”* 的警告对话框，询问您是否绝对确定并了解自己在做什么。
    * **删除的效果**：接受第二次警告后，程序会在后台从该 NVR 注销，从应用程序配置中删除其条目，并**自动扫描预设布局列表**，删除为此 NVR 设备生成的所有动态网格。

## 4. 实时视图和视口叠加

程序的主窗口显示实时直播：
* **摄像头网格**：直接显示来自 Hikvision SDK 的 RTSP 流或源。
* **码流质量选择**：通过右键单击摄像机视口，您可以选择**主码流**以获得最高分辨率，或选择**子码流**以减少网络和显卡负载。
* **双击全屏切换**：在任何相机视口上双击鼠标左键，立即将其最大化以填充整个活动窗口区域（单视口全屏）。再次双击可恢复原来的多机位网格布局。
* **自动隐藏顶栏**：当鼠标光标离开其区域时，顶部选项工具栏（topToolBar）可以自动折叠到屏幕的上边缘（此选项可在“设置”->“自动折叠顶栏”*中配置或直接使用顶栏上的图钉图标）。
* **多显示器和辅助窗口**：您可以打开独立的附加（辅助）窗口，以在多个屏幕或显示器上同时运行不同的网格布局。要打开新窗口，请使用 `Ctrl+N` 键盘快捷键，或单击悬停滑动顶部工具栏上的**“新窗口”**按钮。每个窗口都可以配置自己的网格大小和选定的预设布局。为了确保多显示器配置的最大稳定性，所有窗口都严格以主显示器的 90% 分辨率为中心启动。

### 视口叠加按钮
在每个相机图块/视口的右下角，当将鼠标光标悬停在其上时，会显示一个带有五个功能图标的控制面板（取决于 UI 首选项）：
1. **相机图标（快照）**：允许您从相机源中截取屏幕截图。快照直接从解码器帧缓冲区以流的完整原始分辨率保存，避免由于视口图块的当前大小或屏幕分辨率缩放而造成的任何损失。相机图标闪烁橙色 (`#ff7a00`) 正好 1 秒，确认快照保存成功。
2. **播放图标（存档）**：用于快速打开录音存档。单击此按钮会自动启动该特定摄像机的时间线`PlaybackWindow`，**在当前系统时间前 15 分钟**开始播放（方便的快速向后偏移）。
3. **圆形箭头图标（快速播放/迷你播放器）**：直接在摄像机图块内激活最近 30 分钟的摄像机存档记录的快速视口内播放，无需离开实时网格或启动完整存档窗口。
   * **控制面板**：激活时，滑动控制面板会出现在视口底部，其中包含：
     * **播放/暂停**：暂停或恢复播放流。
     * **时间和偏移量显示**：显示当前播放时间戳及其与实时相比的相对年龄（例如，`-12:34`表示12分34秒前）。
     * **交互式时间轴滑块**：允许单击或拖动以在 30 分钟缓冲区内寻找任何位置。可用的录音片段以青色背景突出显示，红色播放头标记当前位置。
     * **速度倍增器**：循环播放速度（`1x`、`2x` 和 `4x`）以快速扫描素材。
     * **关闭 (✕)**：关闭迷你播放器并立即将视口返回到实时源。
   * **可平移缩放支持**：就像在实时视图中一样，您可以使用放大镜工具放大迷你播放器的某个区域，或者按住鼠标中键并拖动以平移。
4. **1:1 图标（原始比例）**：切换像素到像素的视频显示模式。启用后，视频不会拉伸或扭曲以填充图块边界，而是居中并以其原始分辨率显示。当此模式处于活动状态时，按钮背景和“1:1”文本将以明亮的霓虹灯绿松石色突出显示。
5. **放大镜图标（交互式缩放）**：允许您放大视频源的任何特定区域：
   * **激活**：单击图标将其切换到活动状态（绿松石突出显示）。光标改变形状，工具提示指示：*“单击并拖动相机源进行缩放”*。
   * **操作**：左键单击并在直播源上拖动一个矩形选取框区域。视口将自动裁剪和缩放所选区域以填充整个图块。
     * **按比例缩放**：在绘制缩放矩形时按住**Shift** 键可强制将选区锁定为 16:9 的宽高比，严格限制在视口边界内。
     * **平移缩放图像（平移缩放）**：放大后，您可以按住**鼠标中键（滚轮）**并拖动光标，在视频视野周围自由平移放大视图。这适用于实时视图、回放存档和迷你播放器。
   * **重置**：缩放时，放大镜图标会更改其图标（带有减号的红色边框）。单击它会立即重置变焦，返回完整的相机画面。

## 5. 屏幕布局、预设和工具栏

布局允许您组织相机在屏幕上的排列。在**预设**选项卡（星号图标）中，您可以：
* **创建新预设**：使用任意列和行配置（例如 2x2、3x3、4x4）添加您自己的布局。
* **分配摄像机**：单击网格布局中的视口将其选中，然后打开 NVR 摄像机窗口并单击所需摄像机图块上的**“+”（添加）** 按钮。您还可以使用源视口的右键单击上下文菜单 (*“Zamień miejscami”* / *“交换视口”*) 来交换视口位置，然后单击目标视口。

### 顶部栏按钮（顶部工具栏）
顶部滑动工具栏提供了一整套导航和应用程序控制按钮：
1. **关闭窗口（红色✕按钮）**：关闭活动窗口。为了防止意外点击，它会拦截关闭事件并通过对话框提示您确认退出应用程序。
2. **固定按钮**：控制顶部工具栏的自动隐藏行为。当销钉垂直指向（固定状态）时，该条被锁定到位并保持永久可见。当固定针旋转 -45 度（未固定状态）时，当鼠标离开其区域时，该栏会自动向上滑动到视野之外。
3. **全屏（绿色箭头图标）**：立即将活动窗口切换到全屏模式并返回。在全屏模式下，箭头指向内（折叠），在窗口模式下，箭头指向外（展开）。
4. **最小化（青色最小化图标）**：将应用程序窗口最小化到任务栏。恢复它会将窗口返回到其之前的确切状态（例如最大化或全屏）。
5. **⚙️ 选项 (OPCJE)**：切换配置选项窗口。如果窗口已打开，单击此按钮将其关闭。
6. **📺新窗口（现在还可以）**：打开一个新的、独立的、完全可配置的`Auxiliary Window`，非常适合在多个显示器设置中扩展相机布局。
7. **存档**：打开带有活动时间线和日历的空`PlaybackWindow`（录音播放器），允许通过侧边栏列表从任何配置的 NVR 中手动选择流和摄像机通道。
8. **说明 (INSTRUKCJA)**：打开此手册窗口，根据您的活动区域设置加载英语或波兰语的完整用户文档。
9. **📊 STATS (STATYSTYKI) 开关**：切换开关可从屏幕左边缘滑出系统统计监控面板。
10. **网格锁定开关（挂锁）**：打开该开关（以亮橙色突出显示）时，将禁用相邻网格按钮上的网格调整大小操作，从而保护活动相机布局免受意外更改。
11. **网格大小选择器（从 1x1 到 9x9）**：一行九个按钮，可让您立即定义视口的行和列结构（从单个摄像机 1x1 视图到 9x9 布局中的 81 个同步摄像机源）。当前活动尺寸以亮橙色突出显示。
12. **更多选项（三行汉堡菜单）**：打开滑动`Layout & Grid Tools`工具箱的按钮，用于高级网格调整、几何调整和调试选项（详细信息如下）。
13. **预设/视图按钮**：工具栏右侧的动态呈现按钮，代表您配置的可见预设布局（例如*📹 NVR*、*视图 1* 等）。单击按钮立即切换网格。活动视图以明亮的浅绿松石色突出显示。

### 高级网格定制和比率（布局和网格工具）
打开汉堡（更多选项）菜单会弹出一个专门的布局工具箱。要激活其控件：
1. **解锁工具窗格**：切换最顶部的“解锁工具窗格”开关。这是一项明确的安全措施，旨在防止意外更改复杂布局。
2. **自定义窗口划分（F2或按住）：工具箱显示从1x1到9x9的网格划分按钮。一项极其先进的功能是<b>覆盖和编辑分区大小**的能力。如果您用鼠标左键单击并按住任何网格按钮（或将其聚焦并按**F2**），则会出现一个文本框。您可以键入任何自定义或不对称划分（例如 `2x3`、`1x4` 等），然后按 Enter。该按钮会立即重新编程，单击它会将您的自定义布局应用到主视口。
3. **几何比例**：允许强制网格显示为特定的宽高比：
   * **16:9 宽高比**：将网格容器锁定并缩放为宽屏 16:9 格式（现代 IP 摄像机的标准）。
   * **4:3 宽高比**：将网格容器调整为传统的 4:3 比例（常见于传统模拟/IP 摄像机）。
4. **网格操作（合并突出显示的单元格）**：访问非对称单元格合并功能（详细信息请参阅第 9.2 节）。

## 6.系统统计面板（System Stats）

该面板从实时视图屏幕的左边缘滑出，监控计算机的运行状况以及应用程序生成的负载：
* **监控参数**：
  * **CPU / RAM**：主处理器的使用情况（占所有内核的百分比）以及 `kvision` 进程及其相关下载程序子进程直接使用的 RAM。
  * **GPU / VRAM**：显卡核心利用率（以 % 为单位）以及渲染和硬件解码占用的 VRAM 图形内存量（支持使用 `nvidia-smi` 中的 XML 解析器完整列出 GPU 进程）。
  * **网络（Net）**：来自所有活跃实时播放器、存档播放器和录制下载过程的应用程序的实际下载传输速度。
* **多线程（无卡顿）**：进程和 GPU 数据收集在单独的系统线程 (`StatsWorker`) 上运行。这可以防止视频渲染中出现任何微卡顿（无帧丢失）。
* **固定功能**：单击**“固定”**按钮（固定图标）可将面板锁定在其展开状态。
* **美观**：图表具有明亮的霓虹绿边框、图表曲线下的渐变填充以及平衡的 35% 背景透明度，以确保文本可读性。

## 7. 回放存档播放器

单击特定摄像机或录像机旁边的时钟/播放图标即可使用。它允许同时查看来自多个海康威视摄像机的全时间同步的存档录像。

### 时间线和控制：
* **快速启动（15 分钟前）**：从实时视图中打开存档时，播放器会自动从**当前系统时间前 15 分钟**的时刻开始（而不是从午夜开始）。这允许立即查看刚刚发生的事件。
* **导航**：用鼠标左键拖动可以左右滚动时间线。
* **缩放（缩放）**：您可以使用鼠标滚轮（或缩放按钮）平滑地更改时间线比例 - 从查看全天到精确的 10 分钟精确视图。
* **快速缩放快捷键**：底部控制栏具有专用的圆形图标按钮，可立即缩放时间线视图：
  * **“1h”图标**：放大时间线以进行 1 小时跨度的详细检查。
  * **“8h”图标**：缩放时间线以显示 8 小时跨度。
  * **“24 小时”图标**：重置缩放以适应单个屏幕上的全天 24 小时。
  * **中心（目标）图标**：立即将时间线居中，使红色播放指示器恰好位于屏幕中间（替换旧的文本按钮）。
* **日期导航（日历和日期）**：显示日期旁边的控件允许快速跳转：
  * **“<”（前一天）** 和 **„>”（第二天）** 按钮：让您无需打开日历对话框即可立即向后或向前跳转 24 小时。
  * **日历图标**：打开日历弹出窗口以选择特定日期。
  * **刷新图标**：强制重新搜索录音。单击它会扫描所有活动通道的记录可用性片段的本地缓存，并向 NVR 提交新的查询，这对于加载刚刚录制的文件非常有用（替换旧的文本按钮）。
  * **今天（今天的日期/数字）图标**：立即跳回当天（替换旧的文本按钮）。
* **播放速度快捷方式**：
  * **“1x”、“2x”和“4x”图标**：立即更改播放速度倍数。
* **VCR跳转按钮**：
  * **带有圆形箭头的“15”、“45”和“60”图标**：让您快速向后或向前跳跃指定的秒数。
* **录制可用性栏**：代表在录制器磁盘上找到的视频片段的彩色栏呈现在时间线下方。缓存系统可防止它们在拖动时闪烁。
* **自动跟随（指示器跟踪）**：持续监控播放指示器（垂直红线）。如果指示器超出时间线的可见范围，视图将自动滚动到中心。在用户手动拖动指示器期间，此选项会被智能锁定。

### 播放窗口中的相机侧面板
播放器右边缘的垂直侧面列表列出了所有已配置的 NVR 及其摄像机通道：
* **切换频道**：单击列表中的任何摄像机频道会将其添加为时间轴上的活动播放时段（生成视频播放器）。再次单击它会删除该通道。
* **频道上下文菜单**：右键单击播放网格中的活动视频槽会打开一个菜单，用于：
  * 切换视频质量（主码流/子码流）。
  * 从存档播放中关闭/删除活动播放器。

## 8. 下载录音（Downloader）

从“播放存档”窗口，您可以将选定的录音片段作为 MP4 文件直接下载到计算机驱动器中：
1. 单击所选摄像机旁边的下载图标（向下箭头）。
2. 选择时间范围（录制的开始和结束）。
3. 选择目标文件保存位置。
4. 点击**下载**。

### 高级下载功能：
* **顺序分段下载（1GB 部分）**：程序会自动将您的时间范围查询分割成物理文件分段（每个 NVR 驱动器上大约 1GB），然后逐一下载和转换它们（使用直接转换为 `.mp4` 格式的临时 `.pspart` 文件）。这可确保长时间范围内高度稳定的下载，而不会出现内存溢出或 FFmpeg 转换挂起的情况。
* **整体进度可视化**：进度条（明亮的青色）显示相机在所有段上的整体下载进度。进度条上叠加的状态文本显示当前部分和百分比，例如 `Downloading part 1 of 3... 45% (Overall: 15%)`，并具有轮廓样式以保证在任何背景下的易读性。
* **文件名 IP 清理**：视频文件名（以及实时/存档快照）会自动删除 NVR/DVR IP 地址，以保持其干净且可读（例如 `4_Wejscie_glowne_2026-06-15.mp4` 而不是 `<RECORDER_IP>_4_Wejscie...`）。

## 9. 选项窗口中的高级设置和自定义

选项和配置窗口 (`SideBar`) 由六个专用选项卡组成：

### 1.视口详细信息（监视器图标）
显示当前所选网格图块的高级参数。允许您：
* 输入自定义**主要流 URL** (RTSP/ONVIF) 和**辅助备份 URL** 以进行手动配置。
* 切换静音/取消静音所选摄像机源的音频通道。
* 在**FFmpeg Options Override**文本框中输入高级解码器覆盖。
> [!提示]
> 为了通过 RTSP 实现最快的流连接和最大的稳定性，建议的参数为：
> ```ini
> -analyzeduration 0 -probesize 500000 -fflags nobuffer -flags low_delay -rtsp_transport tcp
> ```

### 2.布局和网格工具（滑块图标）
高级屏幕网格自定义选项：
* 快速切换全屏模式。
* **非对称单元格合并（合并突出显示的单元格）**：一个非常先进的布局编辑器。按住**Ctrl**或**Shift**并单击以选择网格上的多个相邻图块，或者通过按住**Shift**并使用**箭头键**导航来使用键盘，然后单击“合并突出显示的单元格”将它们融合到一个更大的视口中。这使您可以创建完全自定义的不对称网格设计（例如，一个巨大的相机面板，周围有较小的源）。

### 3. 录音机（服务器图标）
用于连接 Hikvision NVR/DVR 设备的完整配置管理器（详细描述见第 3 节）。

### 4.预设（星形图标）
管理您保存的网格布局和摄像机分配。允许创建空网格模板、更改其顺序、切换其顶栏可见性（通过“可见”开关）或在当前窗口中激活它们。

### 5. 设置（齿轮图标）
允许调整全局应用程序设置：
* **允许运行多个实例**：选中此框允许启动 KVision 进程的多个并行副本（默认情况下，它将运行限制为单个活动实例）。
* **自动折叠设置**：自定义顶栏和统计面板的滑动动画时序。
* **允许交换视口**：复选框*“允许交换视口位置”*使您能够动态重新排列网格上的摄像机位置（右键单击源图块 -> 选择“交换视口” -> 左键单击目标图块）。
* **右键单击权限**：切换到锁定/解锁右键单击上下文菜单中的交互式控件（启用上下文菜单、允许交换视口、启用“删除相机”选项、允许更改视口设置、启用流主/次质量选择）。
* **自动取消静音**：进入全屏模式时自动取消活动视口的音频流静音。
* **在全屏模式下隐藏光标**：复选框*“在全屏模式下隐藏光标”*会在全屏查看期间短暂不活动后自动隐藏鼠标光标，以确保视图畅通无阻。
* **语言选择**：即时切换界面翻译（系统默认，波兰语，英语）。
* **UI 首选项**：隐藏/显示视口状态标签或控件徽章（例如自动隐藏图块右下角的控件覆盖层，除非悬停）。
* **默认 FFmpeg 选项和批量更新**：允许配置应用于新视口的全局 FFmpeg 选项（推荐：`-analyzeduration 100000 -probesize 500000 -fflags nobuffer -flags low_delay`）。
  * > **[建议]**如果您要从较旧版本的 KVision（v2.4.5 之前）迁移，强烈建议在全局设置中将 `-analyzeduration 0` 更改为 `-analyzeduration 100000`，然后单击“更新所有摄像头”。在没有音轨的流上设置 `0` 会导致切换延迟，因为 FFmpeg 将阻塞等待音频数据包的硬编码 5 秒超时。设置非常低的值（例如，100000 微秒 = 0.1 秒）可以完全解决此问题，同时保持即时切换。
  * **更新所有摄像机按钮**：按钮*“Zaktualizuj wszystkie kamery”*将这些全局选项应用于所有配置的视口。
  * **低延迟标志（`nobuffer`、`low_delay`）**：
    * `-fflags nobuffer` 禁用内部多路分配器数据包缓冲，以消除随着时间的推移而产生的流延迟，但如果您与摄像机的网络连接不稳定，则可能会导致卡顿。
    * `-flags low_delay` 告诉解码器立即输出帧，但可能会在快速移动下使用 B 帧的流中导致短暂的伪影/拖尾。
  * **覆盖复选框**：您可以通过在各个视口设置对话框中选中**“Nie uwzględniaj zmian w globalnych ustawieniach FFMpeg”**，从全局更新或动态默认值中排除特定视口。对于解决相机卡顿问题非常有用。


### 6. 变更日志（时钟/文档图标）
提供交互式时间线，显示 KVision 的完整发布历史记录、更新、错误修复和功能添加，确保您可以直接访问程序更新详细信息。

## 11. 快照和路径配置

该应用程序允许您在实时视图和回放存档模式下从任何相机视口快速捕获高质量快照。

### 拍摄快照：
1. 每个视口的右下角都有一个相机图标覆盖按钮（详见第 4 节）。
2. 单击相机图标捕获帧并将其保存为 JPEG 图像（质量 98 - 几乎无损）。
3. 相机图标闪烁橙色 (`#ff7a00`) 正好 1 秒，确认拍摄成功。
4. **全分辨率**：在播放存档模式下，快照直接从解码器的帧缓冲区以流的完整本机源分辨率保存，无论屏幕上的活动视口大小或显示缩放比例如何。

### 保存路径配置：
1. 转到**设置**选项卡（选项窗口中的齿轮图标）。
2. 在**保存**（“Zapis”）部分下，您可以配置默认路径：
   * **默认快照路径**：保存快照的文件夹（默认为`~/Obrazy/CCTV`）。
   * **默认录制路径**：保存下载的 MP4 视频的文件夹（默认为 `~/Wideo/CCTV`）。
3. 单击 `...` 浏览器按钮将打开操作系统的本机文件夹选择器（KDE 中的 Breeze）。
4. **浏览器按钮行为**：目录选择器精确地在文本字段中输入的路径（如果存在）处打开。如果该字段为空、无效或指向您无权访问的文件夹，则该对话框将退回并在您的主目录 (`~/`) 中打开。

### 用户界面设置 (UI)：
1. 转至**设置**选项卡（选项窗口中的齿轮图标）或打开**选项**（“Opcje”）窗口。
2. 在**用户界面设置**部分中，您可以自定义覆盖在相机kafelki/视口上的元素的可见性：
   * **在视口左上角显示频道状态**（默认启用）- 显示流加载、播放和连接状态信息。
   * **在视口左下角显示摄像机信息**（默认启用）— 显示从海康威视录像机检索到的摄像机名称。
   * **仅在悬停时在视口右下角显示控制图标**（默认启用）- 当鼠标光标位于特定相机视口之外时，自动隐藏控制按钮面板（快照、存档、1:1 像素到像素、区域缩放）。当您将鼠标移到视口上时，图标会立即出现（无需单击），并在离开时消失，从而最大限度地提高相机流的可见性。
   * **仅在悬停时显示信息字段**（默认禁用）- 类似的选项，隐藏视口左上角的状态和左下角的名称，显示完全干净的相机流，除非光标移动到该特定相机图块上。
