# Εγχειρίδιο χρήστη για το KVision

> [!ΣΗΜΕΙΩΣΗ]
> Αυτό το εγχειρίδιο οδηγιών έχει μεταφραστεί και μορφοποιηθεί αυτόματα με τη βοήθεια της Τεχνητής Νοημοσύνης (AI).

*Το *KVision** είναι μια προηγμένη εφαρμογή επιτραπέζιου υπολογιστή κλάσης VMS (Σύστημα Διαχείρισης βίντεο) που έχει σχεδιαστεί για ταυτόχρονη προβολή ζωντανών ροών βίντεο (RTSP/ONVIF) και ενσωμάτωση με συσκευές εγγραφής Hikvision NVR/DVR (τόσο σε λειτουργία Live όσο και σε λειτουργία αρχείου αναπαραγωγής).

Το πρόγραμμα έχει βελτιστοποιηθεί για σταθερότητα, ομαλή λειτουργία (60 FPS) και ελάχιστη κατανάλωση πόρων συστήματος.

## 1. Περιγραφή των ενεργειών του κουμπιού

Αυτή η ενότητα περιγράφει τη σημασία όλων των γραφικών εικονιδίων και κουμπιών που χρησιμοποιούνται στην εφαρμογή.

* {ICON:zoom_in} **Διαδραστικό ζουμ**: Εναλλάσσει τη μεγέθυνση με κλικ και μεταφορά για μια επιλεγμένη περιοχή μαρκίζας.

## 2. Εγκατάσταση και εκκίνηση

### Εγκατάσταση σε Arch Linux (Pacman)
Για να εγκαταστήσετε το πρόγραμμα από το έτοιμο δυαδικό πακέτο, μεταβείτε στον κατάλογο `packaging/arch/` και εκτελέστε:
```bash
sudo pacman -U kvision-2.4.6-2-x86_64.pkg.tar.zst
```
Το πακέτο θα εγκαταστήσει αυτόματα το πρόγραμμα, το αρχείο ενεργοποίησης `.desktop` και τις απαιτούμενες βιβλιοθήκες Hikvision SDK στη διαδρομή συστήματος `/usr/lib/kvision`.

### Μη αυτόματη συλλογή (από τον πηγαίο κώδικα)
Εάν θέλετε να μεταγλωττίσετε το πρόγραμμα με μη αυτόματο τρόπο (π.χ. σε άλλη διανομή Linux) αντί να χρησιμοποιήσετε το έτοιμο πακέτο:

1. Εγκαταστήστε τις απαιτούμενες εξαρτήσεις κατασκευής και χρόνου εκτέλεσης χρησιμοποιώντας τον διαχειριστή πακέτων. Για Arch Linux / CachyOS:
   ```bash
   sudo pacman -S base-devel cmake qt5-declarative qt5-multimedia qt5-quickcontrols qt5-quickcontrols2 qt5-svg qt5-graphicaleffects qt5-tools ffmpeg git qt5-wayland
   ```
2. Διαμορφώστε το έργο χρησιμοποιώντας το CMake:
   ```bash
   cmake -B build -S . -DCMAKE_BUILD_TYPE=Release -DCMAKE_INSTALL_PREFIX=/usr
   ```
3. Μεταγλωττίστε τον κώδικα:
   ```bash
   cmake --build build -j$(nproc)
   ```
4. Εγκαταστήστε την εφαρμογή στο σύστημα:
   ```bash
   sudo cmake --install build
   ```

### Εκκίνηση
Το πρόγραμμα μπορεί να ξεκινήσει από το μενού συστήματος ή πληκτρολογώντας στο τερματικό:
```bash
kvision
```

### Αντιμετώπιση προβλημάτων Κλίμακα συστήματος (KDE Plasma)

Ξεκινώντας από την έκδοση 2.4.4, το KVision εντοπίζει αυτόματα και εφαρμόζει σωστά την κλασματική κλίμακα επιφάνειας εργασίας (π.χ. 125%, 145%, 150%) που έχει διαμορφωθεί στις ρυθμίσεις του KDE Plasma, επιβάλλοντας εγγενείς πολιτικές μετάδοσης υψηλού DPI στο Qt. Η διεπαφή χρήστη θα πρέπει τώρα να κλιμακώνεται σωστά εκτός συσκευασίας χωρίς καμία χειροκίνητη παρέμβαση.

#### Υποστήριξη Native Wayland

Για να διασφαλίσετε την καλύτερη εμπειρία κλιμάκωσης και απόδοση στο Wayland, βεβαιωθείτε ότι έχετε εγκατεστημένο το προαιρετικό πακέτο `qt5-wayland` στο σύστημά σας. Αυτό επιτρέπει στο KVision να τρέχει εγγενώς στο Wayland αντί να επιστρέψει στο XWayland.

Για Arch Linux / Manjaro / CachyOS:
```bash
sudo pacman -S qt5-wayland
```

Για Debian / Ubuntu / Linux Mint:
```bash
sudo apt install qtwayland5
```

#### Μη αυτόματη παράκαμψη κλιμάκωσης (παλαιού τύπου/Επιτροπή)

Εάν εκτελείτε μια παλαιότερη έκδοση, μια ασυνήθιστη διαμόρφωση ή απλά θέλετε να επιβάλετε με μη αυτόματο τρόπο διαφορετικό συντελεστή κλίμακας από το υπόλοιπο σύστημα, μπορείτε ακόμα να παρακάμψετε την κλίμακα της εφαρμογής χρησιμοποιώντας μεταβλητές περιβάλλοντος.

Για παράδειγμα, για να αναγκάσετε την κλιμάκωση **150%**, εκκινήστε την εφαρμογή από το τερματικό με:
```bash
env QT_FONT_DPI=96 QT_SCALE_FACTOR=1.5 kvision
```

*(Μπορείτε να το εφαρμόσετε μόνιμα επεξεργαζόμενοι την εφαρμογή εκκίνησης `kvision.desktop` στον επεξεργαστή μενού του KDE και προσθέτοντας αυτές τις μεταβλητές στο πεδίο "Μεταβλητές περιβάλλοντος").*

## 3. Διαχείριση συσκευών εγγραφής NVR/DVR

Για να διαμορφώσετε τη σύνδεση σε μια συσκευή εγγραφής Hikvision:
1. Ανοίξτε το παράθυρο επιλογών και μεταβείτε στην καρτέλα **Εγγραφές** (εικονίδιο διακομιστή).
2. Εισαγάγετε τα στοιχεία πρόσβασης της συσκευής:
   * **Διεύθυνση IP**: Η διεύθυνση δικτύου της συσκευής εγγραφής.
   * **Θύρα**: Η θύρα δικτύου SDK (η προεπιλογή είναι `8000`).
   * **Όνομα χρήστη**: Όνομα χρήστη (π.χ. `admin`).
   * **Κωδικός πρόσβασης**: Κωδικός πρόσβασης για πρόσβαση στη συσκευή εγγραφής.
3. Κάντε κλικ στην επιλογή **Σύνδεση και ανακάλυψη** (ή **Αποθήκευση και ενημέρωση**).
4. Μετά την επιτυχή σύνδεση, η εφαρμογή θα εντοπίσει αυτόματα όλες τις ενεργές κάμερες (κανάλια) που είναι συνδεδεμένες στο NVR/DVR και θα τις προσθέσει στη λίστα.
5. Κάνοντας κλικ στο κουμπί **Δημιουργία πλέγματος** θα δημιουργηθεί αυτόματα μια διάταξη θύρας προβολής (προκαθορισμένη) που περιέχει όλες τις ενεργές κάμερες από αυτό το NVR σε μια βέλτιστη διάταξη πλέγματος.
6. **Εμφάνιση λιστών κάμερας (NvrCamerasWindow)**: Κάνοντας κλικ στο εικονίδιο της οθόνης υπολογιστή στην κάρτα οποιασδήποτε συσκευής εγγραφής ανοίγει ένα ειδικό παράθυρο που εμφανίζει όλα τα κανάλια κάμερας που εντοπίστηκαν ως διαδραστικά πλακίδια.
7. **Δημιουργία μικρογραφιών (Δημιουργία μικρογραφιών)**: Στο παράθυρο λίστας καμερών NVR, είναι διαθέσιμο ένα κουμπί *"Δημιουργία μικρογραφιών"*. Κάνοντας κλικ σε αυτό δίνει εντολή στην εφαρμογή να τραβήξει μεμονωμένα καρέ από τη δευτερεύουσα ροή κάθε καναλιού στο παρασκήνιο, ορίζοντας τα ως μικρογραφίες φόντου πλακιδίων. Αυτό παρέχει μια γρήγορη οπτική προεπισκόπηση κάθε κάμερας χωρίς την εκκίνηση πλήρους ζωντανής ροής.
8. **Click-and-Add Feature**: Η εφαρμογή δεν υποστηρίζει τη μεταφορά πλακιδίων (drag and drop) από το παράθυρο της λίστας κάμερας στο κύριο πλέγμα. Η αντιστοίχιση κάμερας γίνεται με απλό και αξιόπιστο τρόπο: πρώτα κάντε αριστερό κλικ σε οποιοδήποτε πλακίδιο θύρας προβολής στο πλέγμα της κύριας οθόνης για να το επιλέξετε (θα τονίσει με φωτεινό περίγραμμα) και, στη συνέχεια, κάντε κλικ στο πράσινο κουμπί **"+" (Αντιστοίχιση σε ενεργή θύρα προβολής)** στο επιθυμητό πλακίδιο κάμερας στο παράθυρο της λίστας κάμερας NVR. Η ροή θα φορτώσει αμέσως σε αυτήν την υποδοχή.
9. **Κατάσταση περιόδου λειτουργίας SDK (Ένδειξη κουκκίδας)**: Δίπλα από την IP κάθε συσκευής εγγραφής στη λίστα, υπάρχει μια έγχρωμη κουκκίδα κατάστασης:
   * **Πράσινο (ΣΥΝΔΕΣΗ)**: Αντιπροσωπεύει μια ενεργή περίοδο λειτουργίας Hikvision SDK, η οποία απαιτείται για έλεγχο PTZ, αιτήματα αρχειοθέτησης χρονοδιαγράμματος και λήψη.
   * **Κόκκινο (ΔΕΝ ΕΙΝΑΙ ΣΥΝΔΕΣΗ)**: Δεν έχει δημιουργηθεί ενεργή περίοδος λειτουργίας SDK (π.χ. πριν από το πρώτο αίτημα SDK ή μετά από μη αυτόματη/αυτόματη αποσύνδεση). Λάβετε υπόψη ότι οι ζωντανές ροές RTSP της κάμερας θα συνεχίσουν να αναπαράγονται ανεξάρτητα από την κατάσταση της περιόδου λειτουργίας SDK.
10. **Μετονομασία τοπικής κάμερας**: Σε οποιοδήποτε πλακίδιο κάμερας στη λίστα καμερών NVR, κάντε κλικ στο εικονίδιο **Επεξεργασία** (μολύβι). Αυτό ανοίγει ένα παράθυρο διαλόγου που σας επιτρέπει να ορίσετε ένα προσαρμοσμένο όνομα για την κάμερα. Αυτό το όνομα αποθηκεύεται τοπικά και ενημερώνεται άμεσα στα ζωντανά προγράμματα αναπαραγωγής, στο χρονοδιάγραμμα και στις προβολές πλακιδίων χωρίς τροποποίηση της φυσικής συσκευής NVR. Μπορείτε να το επαναφέρετε στο προεπιλεγμένο όνομα ανά πάσα στιγμή.
11. **Κατάργηση συσκευής εγγραφής από τη λίστα**: Δίπλα σε κάθε διαμορφωμένη συσκευή εγγραφής στη λίστα, υπάρχει ένα κόκκινο κουμπί κάδου απορριμμάτων. Κάνοντας κλικ σε αυτό ξεκινάει ένα πρωτόκολλο ασφαλείας δύο σταδίων που έχει σχεδιαστεί για την αποφυγή τυχαίας διαγραφής:
    * **Βήμα 1 (Επιβεβαίωση)**: Εμφανίζεται ένα παράθυρο διαλόγου με τίτλο *"Επιβεβαίωση διαγραφής NVR"*, το οποίο ρωτά εάν είστε βέβαιοι ότι θέλετε να διαγράψετε τη συσκευή εγγραφής.
    * **Βήμα 2 (Προειδοποίηση)**: Εμφανίζεται ένα δεύτερο παράθυρο διαλόγου προειδοποίησης με τίτλο *"Προειδοποίηση!"*, το οποίο σας ρωτά εάν είστε απολύτως σίγουροι και ενήμεροι για το τι κάνετε.
    * **Επίδραση διαγραφής**: Με την αποδοχή της δεύτερης προειδοποίησης, το πρόγραμμα αποσυνδέεται από αυτό το NVR στο παρασκήνιο, διαγράφει την καταχώρισή του από τη διαμόρφωση της εφαρμογής και **σαρώνει αυτόματα τη λίστα προκαθορισμένων διατάξεων**, αφαιρώντας όλα τα δυναμικά πλέγματα που δημιουργούνται για αυτήν τη συσκευή NVR.

## 4. Επικαλύψεις Live View και Viewport

Το κύριο παράθυρο του προγράμματος εμφανίζει τη ζωντανή ροή:
* **Πλέγμα κάμερας**: Εμφανίζει ροές ή ροές RTSP απευθείας από το Hikvision SDK.
* **Επιλογή ποιότητας ροής**: Κάνοντας δεξί κλικ σε μια θύρα προβολής κάμερας, μπορείτε να επιλέξετε την **Κύρια ροή** για την υψηλότερη ανάλυση ή την **Δεύτερη ροή** για να μειώσετε το φορτίο του δικτύου και της κάρτας γραφικών.
* **Εναλλαγή πλήρους οθόνης με διπλό κλικ**: Κάνοντας διπλό κλικ με το αριστερό κουμπί του ποντικιού σε οποιαδήποτε θύρα προβολής κάμερας, μεγιστοποιείται άμεσα για να γεμίσει ολόκληρη την περιοχή του ενεργού παραθύρου (πλήρης οθόνη μιας θύρας προβολής). Κάνοντας ξανά διπλό κλικ επαναφέρεται η αρχική διάταξη πλέγματος πολλαπλών καμερών.
* **Αυτόματη απόκρυψη επάνω γραμμής**: Η κορυφαία γραμμή εργαλείων επιλογών (topToolBar) μπορεί να συμπτύξει αυτόματα στο επάνω άκρο της οθόνης όταν ο δρομέας του ποντικιού φεύγει από την περιοχή του (αυτή η επιλογή μπορεί να διαμορφωθεί στις Ρυθμίσεις -> *"Αυτόματη σύμπτυξη επάνω γραμμής"* ή απευθείας χρησιμοποιώντας το εικονίδιο καρφίτσας στην επάνω γραμμή).
* **Πολλαπλή οθόνη και βοηθητικά παράθυρα**: Μπορείτε να ανοίξετε ανεξάρτητα πρόσθετα (βοηθητικά) παράθυρα για την εκτέλεση διαφορετικών διατάξεων πλέγματος ταυτόχρονα σε πολλές οθόνες ή οθόνες. Για να ανοίξετε ένα νέο παράθυρο, χρησιμοποιήστε τη συντόμευση πληκτρολογίου `Ctrl+N` ή κάντε κλικ στο κουμπί **"Νέο παράθυρο"** στην επάνω γραμμή εργαλείων με το δείκτη του ποντικιού. Κάθε παράθυρο μπορεί να διαμορφωθεί με το δικό του μέγεθος πλέγματος και επιλεγμένη προκαθορισμένη διάταξη. Για να διασφαλιστεί η μέγιστη σταθερότητα στις διαμορφώσεις πολλαπλών οθονών, όλα τα παράθυρα ξεκινούν αυστηρά με επίκεντρο την κύρια οθόνη στο 90% της ανάλυσής της.

### Κουμπιά επικάλυψης θυρών προβολής
Στην κάτω δεξιά γωνία κάθε πλακιδίου/θυρίδας προβολής κάμερας, εμφανίζεται ένας πίνακας ελέγχου με τέσσερα λειτουργικά εικονίδια όταν τοποθετείτε τον κέρσορα του ποντικιού πάνω του (ανάλογα με τις προτιμήσεις διεπαφής χρήστη):
1. **Εικονίδιο κάμερας (Στιγμιότυπο)**: Σας επιτρέπει να τραβήξετε ένα στιγμιότυπο οθόνης από τη ροή της κάμερας. Το στιγμιότυπο αποθηκεύεται στην πλήρη, εγγενή ανάλυση της ροής απευθείας από το buffer πλαισίου του αποκωδικοποιητή, αποφεύγοντας τυχόν απώλειες λόγω του τρέχοντος μεγέθους του πλακιδίου της θύρας προβολής ή της κλίμακας ανάλυσης οθόνης. Η επιτυχής αποθήκευση στιγμιότυπου επιβεβαιώνεται από το εικονίδιο της κάμερας που αναβοσβήνει με πορτοκαλί χρώμα (`#ff7a00`) για ακριβώς 1 δευτερόλεπτο.
2. **Εικονίδιο αναπαραγωγής (Αρχείο)**: Χρησιμοποιείται για το γρήγορο άνοιγμα του αρχείου εγγραφών. Κάνοντας κλικ σε αυτό το κουμπί ξεκινά αυτόματα η γραμμή χρόνου `PlaybackWindow` για τη συγκεκριμένη κάμερα, ξεκινώντας την αναπαραγωγή **ακριβώς 15 λεπτά πριν από την τρέχουσα ώρα συστήματος** (μια βολική γρήγορη μετατόπιση προς τα πίσω).
3. **Εικονίδιο 1:1 (Εγγενής κλίμακα)**: Εναλλάσσει τη λειτουργία προβολής βίντεο από pixel σε pixel. Όταν είναι ενεργοποιημένο, το βίντεο δεν τεντώνεται ή παραμορφώνεται για να γεμίσει τα όρια του πλακιδίου, αλλά είναι κεντραρισμένο και εμφανίζεται στην αρχική του εγγενή ανάλυση. Όταν αυτή η λειτουργία είναι ενεργή, το φόντο του κουμπιού και το κείμενο "1:1" επισημαίνονται με έντονο νέον-τιρκουάζ.
4. **Εικονίδιο μεγεθυντικού φακού (Διαδραστικό ζουμ)**: Σας επιτρέπει να μεγεθύνετε οποιαδήποτε συγκεκριμένη περιοχή της ροής βίντεο:
   * **Ενεργοποίηση**: Κάνοντας κλικ στο εικονίδιο, γίνεται ενεργή κατάσταση (τιρκουάζ επισήμανση). Ο κέρσορας αλλάζει σχήμα και μια επεξήγηση εργαλείου δίνει οδηγίες: *"Κάντε κλικ και σύρετε στην τροφοδοσία της κάμερας για μεγέθυνση"*.
   * **Λειτουργία**: Κάντε αριστερό κλικ και σύρετε μια ορθογώνια περιοχή πλαισίων πάνω από τη ζωντανή ροή. Η θύρα προβολής θα περικόψει και θα κλιμακώσει αυτόματα την επιλεγμένη περιοχή για να γεμίσει ολόκληρο το πλακίδιο.
     * **Αναλογικό ζουμ**: Κρατήστε πατημένο το πλήκτρο **Shift** ενώ σχεδιάζετε το ορθογώνιο ζουμ για να αναγκάσετε την επιλογή να κλειδώσει σε λόγο διαστάσεων 16:9, αυστηρά περιορισμένη στα όρια της θύρας προβολής.
     * **Μετατόπιση της μεγεθυσμένης εικόνας (Pan Zoom)**: Αφού κάνετε μεγέθυνση, μπορείτε να κρατήσετε πατημένο το **μεσαίο κουμπί του ποντικιού (τροχός κύλισης)** και να σύρετε τον κέρσορα για να μετακινήσετε ελεύθερα τη μεγεθυμένη προβολή γύρω από το οπτικό πεδίο βίντεο. Αυτό λειτουργεί σε Ζωντανή προβολή, Αρχείο Αναπαραγωγής και Μίνι-πρόγραμμα αναπαραγωγής.
   * **Επαναφορά**: Όταν γίνεται μεγέθυνση, το εικονίδιο του μεγεθυντικού φακού αλλάζει το εικονίδιό του (κόκκινο περίγραμμα με σύμβολο μείον). Κάνοντας κλικ σε αυτό γίνεται αμέσως επαναφορά του ζουμ, επιστρέφοντας στην πλήρη τροφοδοσία της κάμερας.

## 5. Διατάξεις οθόνης, προεπιλογές και γραμμή εργαλείων

Οι διατάξεις σάς επιτρέπουν να οργανώσετε τη διάταξη των καμερών στην οθόνη. Από την καρτέλα **Προεπιλογές** (εικονίδιο αστεριού) μπορείτε:
* **Δημιουργία νέων προεπιλογών**: Προσθέστε τη δική σας διάταξη με οποιαδήποτε διαμόρφωση στηλών και γραμμών (π.χ. 2x2, 3x3, 4x4).
* **Εκχώρηση καμερών**: Κάντε κλικ σε μια θύρα προβολής στη διάταξη πλέγματος για να την επιλέξετε, στη συνέχεια ανοίξτε το παράθυρο των καμερών NVR και κάντε κλικ στο κουμπί **"+" (Προσθήκη)** στο επιθυμητό πλακίδιο κάμερας. Μπορείτε επίσης να ανταλλάξετε θέσεις θυρών προβολής χρησιμοποιώντας το δεξί κλικ στο μενού περιβάλλοντος (*“Zamień miejscami”* / *“Εναλλαγή θυρών προβολής”*) της θύρας προβολής προέλευσης και, στη συνέχεια, κάνοντας κλικ στη θύρα προβολής προορισμού.

### Κουμπιά επάνω γραμμής (επάνω γραμμή εργαλείων)
Η επάνω συρόμενη γραμμή εργαλείων παρέχει ένα ολοκληρωμένο σύνολο κουμπιών πλοήγησης και ελέγχου εφαρμογών:
1. **Κλείσιμο παραθύρου (Κόκκινο κουμπί ✕)**: Κλείνει το ενεργό παράθυρο. Για την αποφυγή τυχαίων κλικ, παρεμποδίζει το συμβάν κλεισίματος και σας ζητά με ένα παράθυρο διαλόγου να επιβεβαιώσετε την έξοδο από την εφαρμογή.
2. **Κουμπί καρφιτσώματος**: Ελέγχει τη συμπεριφορά αυτόματης απόκρυψης της επάνω γραμμής εργαλείων. Όταν η ακίδα είναι στραμμένη κατακόρυφα (κατάσταση καρφιτσωμένης), η μπάρα είναι κλειδωμένη στη θέση της και παραμένει μόνιμα ορατή. Όταν η ακίδα περιστραφεί κατά -45 μοίρες (κατάσταση ξεκαρφιτσωμένης), η γραμμή ολισθαίνει αυτόματα προς τα πάνω και δεν φαίνεται όταν το ποντίκι φεύγει από την περιοχή του.
3. **Πλήρης οθόνη (Εικονίδιο πράσινου βέλους)**: Μεταβαίνει αμέσως το ενεργό παράθυρο σε λειτουργία πλήρους οθόνης και πίσω. Σε λειτουργία πλήρους οθόνης, τα βέλη δείχνουν προς τα μέσα (σύμπτυξη) και σε λειτουργία παραθύρου δείχνουν προς τα έξω (ανάπτυξη).
4. **Ελαχιστοποίηση (Εικονίδιο Cyan Minimization)**: Ελαχιστοποιεί το παράθυρο της εφαρμογής στη γραμμή εργασιών. Η επαναφορά του επαναφέρει το παράθυρο στην ακριβή προηγούμενη κατάστασή του (π.χ. μεγιστοποιημένο ή πλήρη οθόνη).
5. **⚙️ OPTIONS (OPCJE)**: Εναλλάσσει το παράθυρο επιλογών διαμόρφωσης. Εάν το παράθυρο είναι ήδη ανοιχτό, κάνοντας κλικ σε αυτό το κουμπί κλείνει.
6. **📺 NEW WINDOW (NOWE OKNO)**: Ανοίγει ένα νέο, ανεξάρτητο και πλήρως διαμορφώσιμο `Auxiliary Window`, ιδανικό για την επέκταση των διατάξεων της κάμερας σε πολλές ρυθμίσεις οθόνης.
7. **ΑΡΧΕΙΟ**: Ανοίγει ένα κενό `PlaybackWindow` (πρόγραμμα αναπαραγωγής ηχογραφήσεων) με ενεργό χρονοδιάγραμμα και ημερολόγιο, επιτρέποντας τις μη αυτόματες επιλογές ροής και καναλιών κάμερας από οποιοδήποτε διαμορφωμένο NVR μέσω της λίστας πλαϊνής γραμμής.
8. **ΟΔΗΓΙΕΣ (INSTRUCCJA)**: Ανοίγει αυτό το παράθυρο του εγχειριδίου, φορτώνοντας την πλήρη τεκμηρίωση χρήστη στα Αγγλικά ή στα Πολωνικά, ανάλογα με την ενεργή τοποθεσία σας.
9. Διακόπτης **📊 STATS (STATYSTYKI)**: Κάντε εναλλαγή του διακόπτη για να σύρετε προς τα έξω τον πίνακα παρακολούθησης Statistics συστήματος από την αριστερή άκρη της οθόνης.
10. **Διακόπτης κλειδώματος πλέγματος (Λουκέτο)**: Διακόπτης που, όταν είναι ενεργοποιημένος (επισημαίνεται με έντονο πορτοκαλί), απενεργοποιεί τις ενέργειες αλλαγής μεγέθους πλέγματος στα διπλανά κουμπιά πλέγματος, προστατεύοντας την ενεργή διάταξη της κάμερας από τυχαίες αλλαγές.
11. **Επιλογείς μεγέθους πλέγματος (από 1x1 έως 9x9)**: Μια σειρά από εννέα κουμπιά που σας επιτρέπει να ορίσετε αμέσως τη δομή σειρών και στηλών της θύρας προβολής σας (από μια προβολή 1x1 μεμονωμένης κάμερας έως 81 ταυτόχρονες τροφοδοσίες κάμερας σε διάταξη 9x9). Το τρέχον ενεργό μέγεθος τονίζεται με έντονο πορτοκαλί.
12. **Περισσότερες επιλογές (Μενού χάμπουργκερ με τρεις γραμμές)**: Κουμπί ανοίγματος της συρόμενης εργαλειοθήκης `Layout & Grid Tools` για προηγμένο συντονισμό πλέγματος, προσαρμογές γεωμετρίας και επιλογές εντοπισμού σφαλμάτων (αναλυτικά παρακάτω).
13. **Κουμπιά Preset/View**: Κουμπιά με δυναμική απόδοση στη δεξιά πλευρά της γραμμής εργαλείων που αντιπροσωπεύουν τις διαμορφωμένες και ορατές προκαθορισμένες διατάξεις σας (π.χ. *📹 NVR*, *Προβολή 1*, κ.λπ.). Κάνοντας κλικ σε ένα κουμπί αλλάζει αμέσως το πλέγμα. Η ενεργή όψη τονίζεται σε έντονο φως-τιρκουάζ.

### Προηγμένη προσαρμογή και αναλογίες πλέγματος (Διάταξη & Εργαλεία Πλέγματος)
Ανοίγοντας το μενού Hamburger (Περισσότερες επιλογές) εμφανίζεται μια εξειδικευμένη εργαλειοθήκη διάταξης. Για να ενεργοποιήσετε τα χειριστήρια του:
1. **Παράθυρο ξεκλειδώματος εργαλείων**: Ενεργοποιήστε το διακόπτη "Ξεκλείδωμα παραθύρου εργαλείων" στο επάνω μέρος. Αυτό είναι ένα ρητό μέτρο ασφαλείας για την αποφυγή τυχαίων αλλαγών σε πολύπλοκες διατάξεις.
2. **Προσαρμοσμένη διαίρεση παραθύρου (F2 ή Πατήστε παρατεταμένα)**: Η εργαλειοθήκη εμφανίζει κουμπιά διαίρεσης πλέγματος από 1x1 έως 9x9. Ένα εξαιρετικά προηγμένο χαρακτηριστικό είναι η δυνατότητα **παράκαμψης και επεξεργασίας μεγεθών διαίρεσης**. Εάν κάνετε κλικ και κρατάτε πατημένο σε οποιοδήποτε κουμπί πλέγματος με το αριστερό κουμπί του ποντικιού (ή το εστιάσετε και πατήσετε **F2**), εμφανίζεται ένα πλαίσιο κειμένου. Μπορείτε να πληκτρολογήσετε οποιαδήποτε προσαρμοσμένη ή ασύμμετρη διαίρεση (όπως `2x3`, `1x4`, κ.λπ.) και να πατήσετε Enter. Το κουμπί επαναπρογραμματίζεται άμεσα και κάνοντας κλικ σε αυτό εφαρμόζεται η προσαρμοσμένη διάταξη στην κύρια θύρα προβολής.
3. **Αναλογίες γεωμετρίας**: Επιτρέπει την επιβολή της εμφάνισης πλέγματος σε συγκεκριμένους λόγους διαστάσεων:
   * **Λόγος διαστάσεων 16:9**: Κλειδώνει και κλιμακώνει το κοντέινερ πλέγματος σε μορφή ευρείας οθόνης 16:9 (πρότυπο για σύγχρονες κάμερες IP).
   * **Λόγος διαστάσεων 4:3**: Προσαρμόζει το κοντέινερ πλέγματος στην παραδοσιακή αναλογία 4:3 (συνήθης στις παλαιού τύπου αναλογικές/IP κάμερες).
4. **Λειτουργίες πλέγματος (Συγχώνευση επισημασμένων κελιών)**: Πρόσβαση στη δυνατότητα συγχώνευσης ασύμμετρων κελιών (αναλυτικά στην Ενότητα 9.2).

## 6. Πίνακας στατιστικών συστήματος (Στατιστικά συστήματος)

Γλιστρώντας προς τα έξω από το αριστερό άκρο της οθόνης Live View, αυτός ο πίνακας παρακολουθεί την υγεία του υπολογιστή και το φορτίο που δημιουργείται από την εφαρμογή:
* **Παρακολουθούμενες παράμετροι**:
  * **CPU / RAM**: Χρήση του κύριου επεξεργαστή (σε % όλων των πυρήνων) και της μνήμης RAM που χρησιμοποιείται απευθείας από τη διαδικασία `kvision` και τις σχετικές υποδιεργασίες προγράμματος λήψης.
  * **GPU / VRAM**: Η χρήση του πυρήνα της κάρτας γραφικών (σε %) και η ποσότητα μνήμης γραφικών VRAM που καταλαμβάνεται από την απόδοση και την αποκωδικοποίηση υλικού (υποστηρίζει την πλήρη λίστα των διεργασιών GPU χρησιμοποιώντας τον αναλυτή XML από `nvidia-smi`).
  * **ΔΙΚΤΥΟ (Δίκτυο)**: Πραγματική ταχύτητα μεταφοράς λήψης της εφαρμογής από όλα τα ενεργά ζωντανά προγράμματα αναπαραγωγής, τα προγράμματα αναπαραγωγής αρχείων και τις διαδικασίες λήψης εγγραφής.
* **Πολλαπλών νημάτων (χωρίς τραυλισμό)**: Η συλλογή δεδομένων διεργασίας και GPU εκτελείται σε ξεχωριστό νήμα συστήματος (`StatsWorker`). Αυτό αποτρέπει τυχόν μικροτραύλισμα στην απόδοση βίντεο (χωρίς πτώση καρέ).
* **Λειτουργία καρφιτσώματος**: Κάνοντας κλικ στο κουμπί **"Καρφίτσωμα"** (εικονίδιο καρφίτσας) ο πίνακας κλειδώνει στην ανεπτυγμένη του κατάσταση.
* **Αισθητική**: Τα γραφήματα διαθέτουν φωτεινά, νέον-πράσινα περιγράμματα, ντεγκραντέ γέμισμα κάτω από την καμπύλη του γραφήματος και ισορροπημένη διαφάνεια φόντου 35% για να διασφαλιστεί η αναγνωσιμότητα του κειμένου.

## 7. Playback Archive Player

Διατίθεται κάνοντας κλικ στο εικονίδιο ρολογιού/αναπαραγωγής δίπλα σε μια συγκεκριμένη κάμερα ή συσκευή εγγραφής. Επιτρέπει την ταυτόχρονη προβολή αρχειοθετημένων εγγραφών από πολλές κάμερες Hikvision σε πλήρη συγχρονισμό.

### Χρονολόγιο και στοιχεία ελέγχου:
* **Γρήγορη εκκίνηση (15 λεπτά πίσω)**: Όταν ανοίγετε το αρχείο από τη ζωντανή προβολή, το πρόγραμμα αναπαραγωγής ξεκινά αυτόματα από μια στιγμή που πέφτει **ακριβώς 15 λεπτά πριν από την τρέχουσα ώρα συστήματος** (αντί να ξεκινά τα μεσάνυχτα). Αυτό επιτρέπει την άμεση προβολή ενός γεγονότος που μόλις συνέβη.
* **Πλοήγηση**: Μπορείτε να κάνετε κύλιση στη γραμμή χρόνου προς τα αριστερά και προς τα δεξιά, σύροντάς την με το αριστερό κουμπί του ποντικιού.
* **Ζουμ (Κλιμάκωση)**: Μπορείτε να αλλάξετε ομαλά την κλίμακα της γραμμής χρόνου με τον τροχό κύλισης του ποντικιού (ή τα κουμπιά Ζουμ) – από προβολή ολόκληρης της ημέρας σε προβολή ακριβείας 10 λεπτών.
* **Συντομεύσεις γρήγορου ζουμ**: Η κάτω γραμμή ελέγχου διαθέτει ειδικά κυκλικά κουμπιά εικονιδίων για άμεση κλιμάκωση της προβολής γραμμής χρόνου:
  * **εικονίδιο "1 ώρα"**: Μεγεθύνει τη γραμμή χρόνου για λεπτομερή έλεγχο σε διάστημα 1 ώρας.
  * **εικονίδιο "8h"**: Μεγεθύνει τη γραμμή χρόνου για να εμφανίσει ένα διάστημα 8 ωρών.
  * **εικονίδιο "24h"**: Επαναφέρει το ζουμ για να χωρέσει το πλήρες 24ωρο σε μία οθόνη.
  * **Εικονίδιο κέντρου (Στόχος)**: Κεντράρει αμέσως τη γραμμή χρόνου, έτσι ώστε η κόκκινη ένδειξη αναπαραγωγής να βρίσκεται ακριβώς στη μέση της οθόνης (αντικαθιστώντας το παλιό κουμπί κειμένου).
* **Ημερομηνία πλοήγησης (Ημερολόγιο & Ημέρες)**: Τα στοιχεία ελέγχου δίπλα από την εμφανιζόμενη ημερομηνία επιτρέπουν γρήγορα άλματα:
  * Κουμπιά **"<" (Προηγούμενη ημέρα)** και **„>" (Επόμενη ημέρα)**: Σας επιτρέπουν να μεταβείτε 24 ώρες προς τα πίσω ή προς τα εμπρός αμέσως χωρίς να ανοίξετε το παράθυρο διαλόγου ημερολογίου.
  * **Εικονίδιο ημερολογίου**: Ανοίγει το αναδυόμενο παράθυρο ημερολογίου για να επιλέξετε μια συγκεκριμένη ημερομηνία.
  * **Εικονίδιο ανανέωσης**: Επιβάλλει μια νέα αναζήτηση εγγραφών. Κάνοντας κλικ σε αυτό σαρώνει την τοπική κρυφή μνήμη των τμημάτων διαθεσιμότητας εγγραφής για όλα τα ενεργά κανάλια και υποβάλλει νέα ερωτήματα στο NVR, το οποίο είναι πολύ χρήσιμο για τη φόρτωση αρχείων που έχουν εγγραφεί μόλις πριν από λίγα λεπτά (αντικαθιστώντας το παλιό κουμπί κειμένου).
  * **Σήμερα (Σημερινή ημερομηνία/αριθμός) εικονίδιο**: Επιστρέφει αμέσως στην τρέχουσα ημέρα (αντικαθιστώντας το παλιό κουμπί κειμένου).
* **Συντομεύσεις ταχύτητας αναπαραγωγής**:
  * **Εικονίδια "1x", "2x" και "4x"**: Αλλάξτε αμέσως τον πολλαπλασιαστή ταχύτητας αναπαραγωγής.
* **κουμπιά μετάβασης βίντεο**:
  * **Εικονίδια "15", "45" και "60" με κυκλικά βέλη**: Σας επιτρέπουν να μεταβείτε γρήγορα προς τα πίσω ή προς τα εμπρός κατά τον καθορισμένο αριθμό δευτερολέπτων.
* **Γραμμές διαθεσιμότητας εγγραφής**: Οι έγχρωμες γραμμές που αντιπροσωπεύουν τα τμήματα βίντεο που βρέθηκαν στο δίσκο της συσκευής εγγραφής αποδίδονται κάτω από τη γραμμή χρόνου. Ένα σύστημα προσωρινής αποθήκευσης τους εμποδίζει να τρεμοπαίζουν κατά τη μεταφορά.
* **Αυτόματη παρακολούθηση (Παρακολούθηση ένδειξης)**: Η ένδειξη αναπαραγωγής (κάθετη κόκκινη γραμμή) παρακολουθείται συνεχώς. Εάν η ένδειξη βγει εκτός του ορατού εύρους της γραμμής χρόνου, η προβολή θα μετακινηθεί αυτόματα στο κέντρο της. Αυτή η επιλογή κλειδώνεται έξυπνα κατά τη χειροκίνητη μεταφορά ένδειξης από το χρήστη.

### Πλαϊνό πλαίσιο κάμερας στο παράθυρο αναπαραγωγής
Μια κατακόρυφη πλαϊνή λίστα στη δεξιά άκρη της συσκευής αναπαραγωγής εμφανίζει όλα τα διαμορφωμένα NVR και τα κανάλια κάμερας τους:
* **Εναλλαγή καναλιών**: Κάνοντας κλικ σε οποιοδήποτε κανάλι κάμερας στη λίστα, το προστίθεται ως ενεργή υποδοχή αναπαραγωγής στη γραμμή χρόνου (δημιουργώντας ένα πρόγραμμα αναπαραγωγής βίντεο). Εάν κάνετε ξανά κλικ σε αυτό, το κανάλι καταργείται.
* **Μενού περιβάλλοντος καναλιού**: Κάνοντας δεξί κλικ σε μια ενεργή υποδοχή βίντεο στο πλέγμα αναπαραγωγής ανοίγει ένα μενού για:
  * Εναλλαγή ποιότητας βίντεο (Κύρια ροή / Δευτερεύουσα ροή).
  * Κλείσιμο/αφαίρεση της ενεργής συσκευής αναπαραγωγής από την αναπαραγωγή αρχείων.

## 8. Λήψη ηχογραφήσεων (Λήψη)

Από το παράθυρο "Αρχείο αναπαραγωγής", μπορείτε να πραγματοποιήσετε λήψη επιλεγμένων τμημάτων εγγραφών απευθείας στη μονάδα δίσκου του υπολογιστή σας ως αρχεία MP4:
1. Κάντε κλικ στο εικονίδιο λήψης (κάτω βέλος) δίπλα στην επιλεγμένη κάμερα.
2. Επιλέξτε το χρονικό εύρος (έναρξη και τέλος της εγγραφής).
3. Επιλέξτε τη θέση αποθήκευσης του αρχείου προορισμού.
4. Κάντε κλικ στην **Λήψη**.

### Προηγμένες δυνατότητες λήψης:
* **Διαδοχική λήψη τμημάτων (τμήματα 1 GB)**: Το πρόγραμμα χωρίζει αυτόματα το ερώτημά σας για το χρονικό εύρος σε φυσικά τμήματα αρχείων (περίπου 1 GB το καθένα στη μονάδα NVR) και τα κατεβάζει και τα μετατρέπει ένα προς ένα (χρησιμοποιώντας προσωρινά αρχεία `.pspart` που μετατρέπονται απευθείας σε μορφή `.mp4`). Αυτό εξασφαλίζει εξαιρετικά σταθερές λήψεις εύρους μεγάλης διάρκειας χωρίς υπερχείλιση μνήμης ή κολλάει η μετατροπή FFmpeg.
* **Οπτικοποίηση συνολικής προόδου**: Η γραμμή προόδου (έντονο γαλαζοπράσινο χρώμα) εμφανίζει τη συνολική πρόοδο λήψης για την κάμερα σε όλα τα τμήματα. Το κείμενο κατάστασης που επικαλύπτεται στη γραμμή προόδου εμφανίζει το τρέχον τμήμα και τα ποσοστά, π.χ., `Downloading part 1 of 3... 45% (Overall: 15%)`, με ένα στυλ περίγραμμα για να εγγυάται την αναγνωσιμότητα σε οποιοδήποτε φόντο.
* **Καθαρισμός IP ονόματος αρχείου**: Τα ονόματα αρχείων βίντεο (και στιγμιότυπα ζωντανής/αρχειοθέτησης) αφαιρούνται αυτόματα από διευθύνσεις IP NVR/DVR για να διατηρούνται καθαρές και αναγνώσιμες από τον άνθρωπο (π.χ. `4_Wejscie_glowne_2026-06-15.mp4` αντί για `<RECORDER_IP>_4_Wejscie...`).

## 9. Προηγμένες ρυθμίσεις και προσαρμογή στο παράθυρο επιλογών

Το παράθυρο επιλογών και διαμόρφωσης (`SideBar`) αποτελείται από έξι αποκλειστικές καρτέλες:

### 1. Λεπτομέρειες θύρας προβολής (Εικονίδιο οθόνης)
Εμφανίζει προηγμένες παραμέτρους του τρέχοντος επιλεγμένου πλακιδίου πλέγματος. Σας επιτρέπει να:
* Πληκτρολογήστε μια προσαρμοσμένη **Κύρια διεύθυνση URL ροής** (RTSP/ONVIF) και μια **Δευτερεύουσα εφεδρική διεύθυνση URL** για μη αυτόματες διαμορφώσεις.
* Εναλλαγή σίγασης/κατάργησης σίγασης του καναλιού ήχου της επιλεγμένης τροφοδοσίας κάμερας.
* Εισαγάγετε προηγμένες παρακάμψεις αποκωδικοποιητή στο πλαίσιο κειμένου **Παράκαμψη επιλογών FFmpeg**.
> [!ΣΥΜΒΟΥΛΗ]
> Για την ταχύτερη σύνδεση ροής και τη μέγιστη σταθερότητα μέσω RTSP, οι συνιστώμενες παράμετροι είναι:
> ```ini
> -analyzeduration 0 -probesize 500000 -fflags nobuffer -flags low_delay -rtsp_transport tcp
> ```

### 2. Εργαλεία διάταξης και πλέγματος (εικονίδιο ρυθμιστών)
Προηγμένες επιλογές προσαρμογής πλέγματος οθόνης:
* Γρήγορη εναλλαγή για λειτουργία πλήρους οθόνης.
* **Ασύμμετρη συγχώνευση κελιών (Συγχώνευση επισημασμένων κελιών)**: Ένα εξαιρετικά προηγμένο πρόγραμμα επεξεργασίας διάταξης. Κρατήστε πατημένο το **Ctrl** ή το **Shift** και κάντε κλικ για να επιλέξετε πολλά παρακείμενα πλακίδια στο πλέγμα ή χρησιμοποιήστε το πληκτρολόγιό σας κρατώντας πατημένο το **Shift** και πλοηγείστε με τα **πλήκτρα βέλους** και, στη συνέχεια, κάντε κλικ στο "Συγχώνευση επισημασμένων κελιών" για να τα ενώσετε σε ένα μεγαλύτερο παράθυρο προβολής. Αυτό σας επιτρέπει να δημιουργήσετε πλήρως προσαρμοσμένα ασύμμετρα σχέδια πλέγματος (π.χ. ένα τεράστιο πάνελ κάμερας με μικρότερες τροφοδοσίες που το περιβάλλουν).

### 3. Συσκευές εγγραφής (Εικονίδιο διακομιστή)
Διαχείριση πλήρους διαμόρφωσης για συνδέσεις με συσκευές Hikvision NVR/DVR (περιγράφεται λεπτομερώς στην Ενότητα 3).

### 4. Προεπιλογές (Εικονίδιο αστεριού)
Διαχειριστής για τις αποθηκευμένες διατάξεις πλέγματος και τις αναθέσεις κάμερας. Επιτρέπει τη δημιουργία κενών προτύπων πλέγματος, την αλλαγή της σειράς τους, την εναλλαγή της ορατότητάς τους στην επάνω γραμμή (μέσω του διακόπτη "Ορατό") ή την ενεργοποίησή τους στο τρέχον παράθυρο.

### 5. Ρυθμίσεις (Εικονίδιο με το γρανάζι)
Επιτρέπει την προσαρμογή των καθολικών ρυθμίσεων εφαρμογής:
* **Να επιτρέπεται η εκτέλεση πολλών παρουσιών**: Η επιλογή αυτού του πλαισίου επιτρέπει την εκκίνηση πολλαπλών παράλληλων αντιγράφων της διαδικασίας KVision (από προεπιλογή, περιορίζει τις εκτελέσεις σε μία μόνο ενεργή παρουσία).
* **Ρυθμίσεις αυτόματης σύμπτυξης**: Προσαρμόζει τον χρονισμό συρόμενων κινούμενων εικόνων για την επάνω γραμμή και τον πίνακα στατιστικών στοιχείων.
* **Να επιτρέπεται η εναλλαγή θυρών προβολής**: Το πλαίσιο ελέγχου *"Να επιτρέπεται η εναλλαγή θέσεων θυρών προβολής"* σάς δίνει τη δυνατότητα να αναδιατάξετε τις θέσεις της κάμερας στο πλέγμα εν κινήσει (Κάντε δεξί κλικ στο πλακίδιο πηγής -> Επιλέξτε "Εναλλαγή θυρών προβολής" -> αριστερό κλικ στο πλακίδιο στόχου).
* **Δικαιώματα δεξί κλικ**: Εναλλαγή για κλείδωμα/ξεκλείδωμα διαδραστικών στοιχείων ελέγχου στο μενού περιβάλλοντος δεξί κλικ (Ενεργοποίηση μενού περιβάλλοντος, Επιτρέπεται η εναλλαγή θυρών προβολής, Ενεργοποίηση επιλογής "Κατάργηση κάμερας", Επιτρέπεται η αλλαγή ρυθμίσεων θύρας προβολής, Ενεργοποίηση επιλογής ποιότητας ροής Κύρια/Δεύτερη ποιότητα ροής).
* **Αυτόματη κατάργηση σίγασης**: Καταργεί αυτόματα τη σίγαση της ροής ήχου της ενεργής θύρας προβολής κατά την είσοδο σε λειτουργία πλήρους οθόνης.
* **Απόκρυψη δρομέα σε πλήρη οθόνη**: Το πλαίσιο ελέγχου *"Απόκρυψη δρομέα σε λειτουργία πλήρους οθόνης"* αποκρύπτει αυτόματα τον κέρσορα του ποντικιού μετά από μια σύντομη περίοδο αδράνειας κατά την προβολή πλήρους οθόνης για να διασφαλιστεί η ανεμπόδιστη προβολή.
* **Επιλογή γλώσσας**: Εναλλάσσει άμεσα τη μετάφραση της διεπαφής (Προεπιλογή συστήματος, Πολωνικά, Αγγλικά).
* **Προτιμήσεις διεπαφής χρήστη**: Απόκρυψη/εμφάνιση ετικετών κατάστασης θυρών προβολής ή σημάτων ελέγχου (όπως η αυτόματη απόκρυψη των επικαλύψεων ελέγχου στην κάτω δεξιά γωνία των πλακιδίων, εκτός εάν τοποθετείται το δείκτη του ποντικιού).
* **Προεπιλογές FFmpeg & Μαζική ενημέρωση**: Επιτρέπει τη διαμόρφωση των καθολικών επιλογών FFmpeg που εφαρμόζονται σε νέες θύρες προβολής (συνιστάται: `-analyzeduration 100000 -probesize 500000 -fflags nobuffer -flags low_delay`).
  * > **[ΣΥΣΤΑΣΗ]** Εάν πραγματοποιείτε μετεγκατάσταση από μια παλαιότερη έκδοση του KVision (πριν από την έκδοση 2.4.5), συνιστάται ιδιαίτερα να αλλάξετε το `-analyzeduration 0` σε `-analyzeduration 100000` στις καθολικές ρυθμίσεις και να κάνετε κλικ στην επιλογή "Ενημέρωση όλων των καμερών". Η ρύθμιση `0` σε ροές χωρίς ηχητικό κομμάτι προκαλεί καθυστερήσεις στην εναλλαγή επειδή το FFmpeg θα μπλοκάρει για ένα σκληρό κωδικοποιημένο χρονικό όριο 5 δευτερολέπτων αναμονής για πακέτα ήχου. Η ρύθμιση μιας πολύ χαμηλής τιμής (π.χ. 100000 μικροδευτερόλεπτα = 0,1 δευτ.) επιλύει πλήρως αυτό το ζήτημα, διατηρώντας παράλληλα την άμεση εναλλαγή.
  * **Κουμπί ενημέρωσης όλων των καμερών**: Το κουμπί *„Zaktualizuj wszystkie kamery”* εφαρμόζει αυτές τις καθολικές επιλογές σε όλες τις διαμορφωμένες θύρες προβολής.
  * **Σημαίες χαμηλής καθυστέρησης (`nobuffer`, `low_delay`)**:
    * Το `-fflags nobuffer` απενεργοποιεί την εσωτερική προσωρινή αποθήκευση πακέτων demuxer για να εξαλείψει την καθυστέρηση ροής με την πάροδο του χρόνου, αλλά μπορεί να προκαλέσει τραυλισμό εάν η σύνδεση δικτύου σας με την κάμερα είναι ασταθής.
    * Το `-flags low_delay` λέει στον αποκωδικοποιητή να εξάγει καρέ αμέσως, αλλά μπορεί να προκαλέσει σύντομα τεχνουργήματα/κηλίδες σε ροές που χρησιμοποιούν πλαίσια B υπό γρήγορες κινήσεις.
  * **Παράκαμψη πλαισίου ελέγχου**: Μπορείτε να εξαιρέσετε συγκεκριμένες θύρες προβολής από καθολικές ενημερώσεις ή δυναμικές προεπιλογές επιλέγοντας **"Nie uwzględniaj zmian w globalnych ustawieniach FFMpeg"** στο μεμονωμένο παράθυρο διαλόγου Ρυθμίσεις θυρών προβολής. Χρήσιμο για την αντιμετώπιση προβλημάτων με κάμερες που τραυλίζουν.


### 6. Καταγραφή αλλαγών (Εικονίδιο ρολογιού/εγγράφου)
Παρουσιάζει ένα διαδραστικό χρονοδιάγραμμα που δείχνει το πλήρες ιστορικό κυκλοφορίας, τις ενημερώσεις, τις διορθώσεις σφαλμάτων και τις προσθήκες λειτουργιών του KVision, διασφαλίζοντας ότι έχετε άμεση πρόσβαση στις λεπτομέρειες των ενημερώσεων του προγράμματος.

## 10. Συντομεύσεις πληκτρολογίου και χειριστήρια ποντικιού

### Συντομεύσεις πληκτρολογίου:
| Πλήκτρο / Συντόμευση | Δράση |
|---|---|
| **F** / **F11** | Εναλλαγή λειτουργίας πλήρους οθόνης. |
| **Μ** | Σίγαση / κατάργηση σίγασης ήχου (λειτουργεί για την ενεργή κάμερα με ήχο). |
| **Διάστημα** | Αναπαραγωγή / Παύση αναπαραγωγής στο παράθυρο Αρχείο Αναπαραγωγής. |
| **Alt + 1** έως **Alt + 9** | Γρήγορη μετάβαση σε προεπιλογή/διάταξη στους δείκτες 1 έως 9. |
| **Alt + Αριστερό βέλος** | Γρήγορη εναλλαγή στην προηγούμενη προεπιλογή/διάταξη της συλλογής. |
| **Alt + Δεξί βέλος** | Γρήγορη εναλλαγή στην επόμενη προεπιλογή/διάταξη της συλλογής. |
| **Πλήκτρα βέλους (Πάνω/Κάτω/Αριστερά/Δεξιά)** | Πλοηγηθείτε και μετακινήστε την ενεργή εστίαση/επιλογή μεταξύ των θυρών προβολής της κάμερας. |
| **Shift + Πλήκτρα βέλους** | Επιλέξτε πολλές παρακείμενες θύρες προβολής κάμερας ταυτόχρονα (χρησιμοποιούνται για συγχώνευση κελιών, κ.λπ.). |
| **Ctrl + N** | Ανοίξτε ένα νέο, ανεξάρτητο βοηθητικό παράθυρο. |
| **+** / **-** | Zoom in / Zoom out (κάμερες Hikvision με δυνατότητα PTZ). |
| **Shift** (μεταφορά σύρματος) | Υποχρεώστε την επιλογή πλαισίων να κλειδώσει σε λόγο διαστάσεων 16:9 κατά το διαδραστικό ζουμ. |
| **Esc** | Έξοδος από τη λειτουργία πλήρους οθόνης / ακυρώστε την ενεργή επιλογή θύρας προβολής. |

### Αλληλεπίδραση ποντικιού:
* **Αριστερό κουμπί του ποντικιού**:
  * Το **διπλό κλικ** σε μια θύρα προβολής κάμερας στο πλέγμα μεγιστοποιεί σε πλήρη οθόνη. Ένα άλλο διπλό κλικ επαναφέρει την προβολή πλέγματος.
  * Σύρετε τη γραμμή χρόνου στο παράθυρο Αναπαραγωγή για πλοήγηση.
* **Μεσαίο κουμπί του ποντικιού (κλικ και σύρσιμο τροχού κύλισης)**:
  * Σύρετε για μετατόπιση και μετακινήστε την ψηφιακά μεγέθυνση/μεγέθυνση εικόνας γύρω από το οπτικό πεδίο της θύρας προβολής (λειτουργεί σε Live, Archive και Mini-player).
* **Δεξί κουμπί του ποντικιού (Μενού περιβάλλοντος)**:
  * Ανοίγει ένα μενού γρήγορων ρυθμίσεων για την επιλεγμένη θύρα προβολής (επιτρέπει την αφαίρεση της κάμερας από το πλέγμα, την αλλαγή μεταξύ Κύριων/Δευτερευουσών ροών ή την πρόσβαση σε μεμονωμένες παραμέτρους οθόνης).
* **Τροχός κύλισης ποντικιού**:
  * Προσαρμόζει την κλίμακα (Ζουμ) της γραμμής χρόνου στο πρόγραμμα αναπαραγωγής αρχείων αναπαραγωγής.

## 11. Λήψη στιγμιότυπων και διαμόρφωση διαδρομής

Η εφαρμογή σάς επιτρέπει να τραβήξετε γρήγορα στιγμιότυπα υψηλής ποιότητας από οποιοδήποτε παράθυρο προβολής κάμερας τόσο σε λειτουργία Live View όσο και σε λειτουργία Archive Playback.

### Λήψη στιγμιότυπων:
1. Ένα κουμπί επικάλυψης εικονιδίου κάμερας είναι διαθέσιμο στην κάτω δεξιά γωνία κάθε θύρας προβολής (αναλυτικά στην Ενότητα 4).
2. Κάνοντας κλικ στο εικονίδιο της κάμερας καταγράφει το καρέ και το αποθηκεύει ως εικόνα JPEG (ποιότητα 98 - σχεδόν χωρίς απώλειες).
3. Η επιτυχής λήψη επιβεβαιώνεται από το εικονίδιο της κάμερας που αναβοσβήνει με πορτοκαλί χρώμα (`#ff7a00`) για ακριβώς 1 δευτερόλεπτο.
4. **Πλήρης ανάλυση**: Στη λειτουργία Αρχείο αναπαραγωγής, τα στιγμιότυπα αποθηκεύονται στην πλήρη ανάλυση της εγγενούς πηγής της ροής απευθείας από το buffer πλαισίου του αποκωδικοποιητή, ανεξάρτητα από το μέγεθος της ενεργής θύρας προβολής στην οθόνη ή την κλιμάκωση της οθόνης.

### Αποθήκευση διαμορφώσεων διαδρομής:
1. Μεταβείτε στην καρτέλα **Ρυθμίσεις** (εικονίδιο με το γρανάζι στο παράθυρο επιλογών).
2. Στην ενότητα **Αποθήκευση** ("Zapis"), μπορείτε να διαμορφώσετε τις προεπιλεγμένες διαδρομές:
   * **Προεπιλεγμένη διαδρομή στιγμιότυπων**: Φάκελος όπου θα αποθηκευτούν τα στιγμιότυπα (προεπιλογή σε `~/Obrazy/CCTV`).
   * **Προεπιλεγμένη διαδρομή εγγραφών**: Φάκελος όπου θα αποθηκευτούν τα ληφθέντα βίντεο MP4 (προεπιλογή σε `~/Wideo/CCTV`).
3. Κάνοντας κλικ στο κουμπί του προγράμματος περιήγησης `...` ανοίγει ο επιλογέας εγγενούς φακέλου του λειτουργικού σας συστήματος (Breeze στο KDE).
4. **Συμπεριφορά κουμπιού προγράμματος περιήγησης**: Ο επιλογέας καταλόγου ανοίγει ακριβώς στη διαδρομή που πληκτρολογείται στο πεδίο κειμένου (αν υπάρχει). Εάν το πεδίο είναι κενό, μη έγκυρο ή δείχνει σε έναν φάκελο στον οποίο δεν έχετε πρόσβαση, το παράθυρο διαλόγου επιστρέφει και ανοίγει στον αρχικό σας κατάλογο (`~/`).

### Ρυθμίσεις διεπαφής χρήστη (UI):
1. Μεταβείτε στην καρτέλα **Ρυθμίσεις** (εικονίδιο με το γρανάζι στο παράθυρο επιλογών) ή ανοίξτε το παράθυρο **Επιλογές** ("Opcje").
2. Στην ενότητα **Ρυθμίσεις διεπαφής χρήστη**, μπορείτε να προσαρμόσετε την ορατότητα των στοιχείων που επικαλύπτονται στο kafelki/viewports της κάμερας:
   * **Εμφάνιση κατάστασης καναλιού στην επάνω αριστερή γωνία της θύρας προβολής** (η προεπιλογή είναι ενεργοποιημένη) — Εμφανίζει πληροφορίες φόρτωσης ροής, αναπαραγωγής και κατάστασης σύνδεσης.
   * **Εμφάνιση πληροφοριών κάμερας στην κάτω αριστερή γωνία της θύρας προβολής** (η προεπιλογή είναι ενεργοποιημένη) — Εμφανίζει το όνομα της κάμερας που ανακτήθηκε από τη συσκευή εγγραφής Hikvision.
   * **Εμφάνιση εικονιδίων ελέγχου στην κάτω δεξιά γωνία της θύρας προβολής μόνο όταν τοποθετείται το δείκτη του ποντικιού** (η προεπιλογή είναι ενεργοποιημένη) — Αποκρύπτει αυτόματα τον πίνακα κουμπιών ελέγχου (στιγμιότυπο, αρχειοθέτηση, 1:1 pixel-to-pixel, ζουμ περιοχής) όταν ο δρομέας του ποντικιού βρίσκεται εκτός της συγκεκριμένης θύρας προβολής της κάμερας. Τα εικονίδια εμφανίζονται αμέσως μόλις μετακινήσετε το ποντίκι σας πάνω από τη θύρα προβολής (δεν απαιτείται κλικ) και εξαφανίζονται όταν φεύγετε, μεγιστοποιώντας την ορατότητα των ροών της κάμερας σας.
   * **Εμφάνιση πεδίων πληροφοριών μόνο όταν τοποθετείτε το δείκτη του ποντικιού** (προεπιλογή απενεργοποιημένη) — Ανάλογη επιλογή που αποκρύπτει την κατάσταση στην επάνω αριστερή γωνία και το όνομα στην κάτω αριστερή πλευρά των θυρών προβολής, εμφανίζοντας μια εντελώς καθαρή ροή κάμερας, εκτός εάν ο κέρσορας μετακινηθεί πάνω από το συγκεκριμένο πλακίδιο κάμερας.
